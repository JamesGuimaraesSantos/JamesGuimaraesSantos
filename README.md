### Olá! O meu nome é James
- 🤓 Sou aluno de pós-graduação em Física.
- 🔭 No meu projeto de mestrado, pesquiso na área de Física das Partículas Elementares e Campos. O foco do meu trabalho é a Correspondência ODE/IM para modelos de 19 vértices. Estudo os métodos ODE/IM para resolver as equações de Yang-Baxter, o que permite calcular autovalores exatos da energia. Estes problemas se conectam à correspondência holográfica AdS/CFT, ligando-se ao espectro de massas de partículas, como glueballs e mésons pesados, em modelos de interações fortes.
